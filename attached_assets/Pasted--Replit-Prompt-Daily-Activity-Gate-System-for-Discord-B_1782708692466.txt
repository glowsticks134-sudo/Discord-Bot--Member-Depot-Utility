# Replit Prompt – Daily Activity Gate System for Discord Bot

I want you to build a brand-new feature for my Discord bot called **Daily Activity Access**. This feature should be production-ready, efficient, and easy to configure.

## Goal

Members must send **35 messages every day** to unlock access to our farming/service channels.

Once they reach 35 valid messages:

* They automatically receive a configurable role (example: `Daily Active`).
* That role grants access to specific channels (such as the Farm Channel and Bot Service channels).
* They receive a congratulatory DM and a message in the server letting them know they've unlocked access for the rest of the day.

At the end of each day:

* Every user's daily message count resets to 0.
* The access role is automatically removed from everyone.
* Members must earn it again the next day.

---

# Requirements

## Message Tracking

Track messages per user.

Only count messages that:

* Are sent by real users (ignore bots).
* Are at least 5 characters long.
* Are not commands.
* Are not repeated spam.
* Ignore rapid spam (example: multiple messages within 2 seconds).
* Ignore duplicate messages.

Store:

* User ID
* Guild ID
* Daily Message Count
* Last Message
* Last Message Time
* Date

---

## Daily Requirement

Requirement:
35 valid messages

When a user reaches 35:

Automatically:

* Give configured role
* Unlock farming channels through Discord permissions
* Send DM:

🎉 Congratulations!
You completed today's activity requirement.
You now have access to today's farming channels.

Also send a server notification:

✅ @User has unlocked today's farming access!

---

## Daily Reset

Automatically reset every day at **12:00 AM** using the server's configured timezone.

Reset:

* Daily message counts
* Remove Daily Active role from every member
* Lock access again

No manual commands should be required.
